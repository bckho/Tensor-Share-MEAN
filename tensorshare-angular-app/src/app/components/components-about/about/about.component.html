About TensorShare

TensorShare is an app that allows users to create their own supervised learning models for machine learning and share among other users. Users can view created models from other users and place comments on the model. Users can also create issues for a model in order to indicate that there's a problem with the provided model and could provide solutions and improve the supervised model.
